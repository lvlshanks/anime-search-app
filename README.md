## Tech Stack

React v17, MaterialUI, Typescript, axios, react-router-dom


## Run Locally

Clone the project

```bash
  git clone https://github.com/lvlshanks/anime-search-app.git
```

Go to the project directory

```bash
  cd anime-search-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
